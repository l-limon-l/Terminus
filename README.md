## 🇬🇧 English

**Terminus** is a sleek, developer-first HTML/CSS landing page template with a terminal-inspired aesthetic. Built with vanilla web technologies, it features a built-in language toggler, smooth micro-animations, and a suite of ready-to-use secondary pages. Perfect for tech products, CLI tools, developer utilities, or cybersecurity startups.

### 🚀 Features
- **Developer-First Aesthetic**: Clean, terminal-like UI with monospace fonts (JetBrains Mono) and mint-green "success" accents.
- **Zero Dependencies**: Built with 100% Vanilla HTML, CSS, and JS. No Webpack, no npm, no complex build steps required. Just open and edit!
- **Built-in Localization (i18n)**: Features a lightweight, vanilla JS language toggler (English & Russian included).
- **Ready-to-use Pages**: Comes with pre-formatted secondary pages (`roadmap.html`, `privacy.html`, `docs.html`, etc.).
- **Fully Responsive**: Carefully optimized to look perfect on desktops, tablets, and smartphones.

### 🛠️ Tech Stack
- HTML5
- Vanilla CSS3 (Custom Properties, Flexbox, CSS Grid)
- Vanilla JavaScript
- JetBrains Mono & Inter Fonts

### 📦 Installation & Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/terminus.git
   ```
2. Double-click `index.html` or open it in your browser of choice.
3. **Customization**: Terminus uses CSS variables. Open `assets/css/styles.css` to easily change the primary mint color to your brand color. Edit the HTML files directly to change text (look for `[Square Brackets]`).

## 🇷🇺 Русский

**Terminus** — это стильный HTML/CSS шаблон лендинга для разработчиков с эстетикой терминала. Построенный на "чистых" веб-технологиях, он включает встроенный переключатель языков, плавные микроанимации и набор готовых второстепенных страниц. Идеально подходит для технических продуктов, CLI-утилит или стартапов по кибербезопасности.

### 🚀 Особенности
- **Эстетика для разработчиков**: Чистый UI, напоминающий терминал, с моноширинными шрифтами (JetBrains Mono) и мятно-зелеными акцентами.
- **Ноль зависимостей**: 100% Vanilla HTML, CSS и JS. Никакого Webpack или npm. Просто открывай и редактируй!
- **Встроенная локализация (i18n)**: Легковесный переключатель языков на чистом JS (английский и русский в комплекте).
- **Готовые страницы**: Шаблон включает сверстанные страницы (roadmap, privacy, docs и т.д.).
- **Адаптивность**: Идеально смотрится на любых устройствах.

### 🛠️ Стек технологий
- HTML5
- Vanilla CSS3 (Custom Properties, Flexbox, CSS Grid)
- Vanilla JavaScript
- Шрифты JetBrains Mono и Inter

### 📦 Установка и запуск
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/terminus.git
   ```
2. Откройте файл `index.html` в браузере.
3. **Настройка**: Шаблон использует CSS-переменные. Откройте `assets/css/styles.css`, чтобы изменить мятный цвет на ваш корпоративный. Тексты меняются прямо в HTML-файлах (ищите плейсхолдеры в `[квадратных скобках]`).
